# Bada-Lang

A hybrid systems programming language combining C structure with Rust-style safety. Compiles to native ARM64 binaries via LLVM IR, with full C/C++ interoperability through an integrated C++ transpiler mode.

## Features

- Native compilation via LLVM IR (ARM64, x86_64, RISC-V)
- Automatic C++ transpiler mode for full C/C++ interop
- Inline assembly support across multiple architectures
